# PROMPT REPLIT v4 — copy semua di bawah garis ini

---

Rebuild this project **from scratch** as a polished, single-page **interactive presentation website** for **AIGYPT × AINA**, entirely in **Bahasa Indonesia**. Audience: **khalayak umum** — a live crowd of up to 10,000 people of all backgrounds (students, community elders, business owners), most of whom know nothing about AI. The site is structured as a **30-minute stage talk divided into 4 chapters ("Bab")**, works like a keyboard-driven slide deck with step-by-step reveals, and doubles as a normal responsive website afterward.

The writing must be warm, simple, and story-driven. Zero tech jargon without explanation. Every slide has one idea only.

## Design rules (strict — previous builds failed by over-decorating)
1. **ONE font: "Plus Jakarta Sans"** (Google Fonts; 400, 500, 600, 700, 800). Headlines weight 800, letter-spacing -0.02em. Eyebrows/labels: weight 600, uppercase, letter-spacing 0.18em, 12–13px.
2. **Palette — black with a hint of purple, exactly these tokens:**
   - `--bg:#0B0712` · `--surface:#151021` · `--border:rgba(167,139,250,.14)`
   - `--accent:#7C3AED` · `--accent-soft:#A78BFA` · `--text:#F4F1FA` · `--muted:#9D93B8`
   - Only gradient allowed: `linear-gradient(120deg,#A78BFA,#7C3AED)` — for one key word per headline, the primary button, and the progress bar. **No cyan, no magenta, no animated background.**
3. One decorative element only: a static radial purple glow behind slide 1 (`radial-gradient(ellipse at 50% 25%, rgba(124,58,237,.22), transparent 60%)`).
4. Max content width 1120px, generous whitespace, all animations ≤500ms using only transform/opacity, `prefers-reduced-motion` respected.

## Logos (real files in `/images` — never draw logos with CSS)
- `images/logo-aigypt.png` (purple, transparent) → nav (~34px), hero (~110px), favicon.
- `images/logo-aigypt-white.png` → footer.
- `images/logo-aina.png` (transparent) → AINA reveal slide + chat avatar. **If the file is missing, fall back** to a rounded gradient square with "AI" in weight 800 — never a broken-image icon.

## Presentation system
- Full-viewport slides (`min-height:100svh`), `scroll-snap-type:y mandatory` on desktop; below 768px snap is off and it behaves as a normal scrolling site (dots hidden).
- **Keyboard:** `Space`/`ArrowDown`/`ArrowRight` → next fragment, or next slide when fragments are done. `ArrowUp`/`ArrowLeft` → back. `Home`/`End` → first/last. `F` fullscreen. `T` presenter timer. `R` replays the AINA chat. Click/tap also advances.
- **Fragments:** marked elements start hidden (`opacity:0; translateY(14px)`) and reveal one per keypress (400ms). On mobile scroll, fragments simply appear.
- **Chapter system:** slides belong to 4 chapters. A small fixed label top-center shows the current chapter (e.g. `BAB 3 · AINA`). The right-side slide dots are **grouped by chapter** with a small gap between groups. Each chapter opens with a divider slide.
- **Presenter timer (T):** hidden pill top-left, `mm:ss / 30:00`, starts on first keypress, amber at 25:00, red past 30:00.
- Tiny muted label bottom-left of each slide: `±N MENIT`.
- Thin gradient progress bar top.
